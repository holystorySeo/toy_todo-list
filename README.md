# toy_todo-list
이 토이 프로젝트는 유튜브 채널 <Dev Ed> 의 Beginner Vanilla Javascript Project Tutorial 를 보고 따라한 것입니다.
현재는 JS로만 구성되어 있으나 코드 리팩토링을 통해 Reat로 리팩토링할 예정입니다.
